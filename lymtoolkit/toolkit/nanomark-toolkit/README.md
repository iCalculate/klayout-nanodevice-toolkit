# NanoMark Toolkit

Install:
1. Run `lymtoolkit\install_lymtoolkit.bat`
2. Restart KLayout
3. Open `Tools -> NanoMark -> NanoMark GUI`
4. Or click the `NanoMark` toolbar button

Included:
- `EBL Writefield Mark` GUI
- `General Mark Array` GUI
- Live preview with pan / zoom
- Import / export parameter JSON
- Insert generated mark geometry into the active KLayout cell
